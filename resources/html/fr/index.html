Aide

Comment je joue?
Déplacez les pièces en les cliquant pour les placer dans le bon ordre. Seules les pièces adjacentes au vide peuvent être déplacées.
Comment changer la langue?
Cliquez sur l'icône des paramètres à la droite et choisissez votre langue.
Comment sauvegarder?
Cliquez sur l'icône de sauvegarde à la droite pour sauver votre jeu.
Comment fonctionne le pointage?
Lorsque vous terminez, votre temps et vos déplacements sont sauvegardés.
